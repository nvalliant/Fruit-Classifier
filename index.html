About

What is RIPE.AI?

A computer vision project built for LE01 Semester 4 — Binus University

🎯

Objective

Automatically classify fruit ripeness into three stages — unripe, ripe, and rotten — to support quality control and reduce food waste.

🧪

Approach

Traditional computer vision pipeline using HSV color analysis and GLCM texture features, classified by a Support Vector Machine (SVM) with RBF kernel across 9 classes (3 fruits × 3 stages).

Technology

The CV Pipeline

Traditional computer vision — no deep learning required

01
Image Preprocessing
Resize to 128×128 · Gaussian blur (5×5) · Otsu's thresholding for fruit segmentation from background
02
HSV Color Features
Extract mean & std of Hue, Saturation, Value channels from masked fruit pixels — captures ripeness-related color transitions
03
GLCM Texture Features
Gray-Level Co-occurrence Matrix at distance=5, angles=[0°, 45°] — extracts contrast, correlation, energy, and homogeneity
04
SVM Classification
StandardScaler normalization → SVC with RBF kernel, One-vs-Rest strategy · GridSearchCV tuned: C=[0.1,1,10], γ=[scale,0.001] · 9-class output (3 fruits × 3 stages)
